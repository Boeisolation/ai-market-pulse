# Roadmap

AI Market Pulse is focused on becoming a lightweight AI portfolio research site, not a full brokerage or trading engine.

## Near Term

- Polish public README with screenshots and a demo GIF.
- Add GitHub Pages publishing guide with screenshots.
- Add more robust A-share provider fallback and data freshness warnings.
- Add portfolio CSV examples for common brokers.

## Mid Term

- Add local web dashboard controls for filtering, symbols, and date ranges.
- Add earnings/dividend/macro event calendar integrations.
- Add AI cache inspection and invalidation commands.
- Add optional sentiment and filings/transcripts inputs.

## Long Term

- Add plugin-like data provider registry.
- Add multi-portfolio workspaces.
- Add team publishing mode with private/public report split.
- Add hosted demo site and release automation.

## Non-Goals

- No automated order placement.
- No guaranteed return claims.
- No opaque recommendation engine that tells users to buy or sell.
