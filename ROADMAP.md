# Roadmap

AI Market Pulse is a lightweight, transparent AI + quant research cockpit. It stays rules-first, reproducible, broker-independent, and deployable without a permanent server.

## Shipped In The Current Release Candidate

- Arbitrary symbol analysis from the local visual console and CLI.
- Editable AI-assisted brokerage screenshot import.
- Tag-driven theme research in reports and the history dashboard.
- Report-grounded single-turn AI Q&A.
- Plugin-style market data provider registry with ordered fallback.
- Deduplicated threshold alerts with optional GitHub Actions scheduling.
- Benchmark-relative strength, freshness checks, portfolio attribution, and bilingual static publishing.

## Next

- Add earnings, dividend, and macro event calendars.
- Add AI cache inspection and invalidation controls to the visual console.
- Add multi-portfolio workspaces with named local configurations.
- Add optional filings, transcripts, and sentiment inputs with explicit provenance.
- Add broker-specific screenshot fixtures and extraction regression tests without storing private user data.

## Later

- Add deterministic factor backtesting with transaction-cost assumptions and benchmark comparison.
- Add team publishing with private/public report splits.
- Add provider health telemetry, quotas, and richer capability discovery.
- Add hosted demo and automated semantic releases.

## Non-Goals

- No automated order placement or broker connection.
- No guaranteed return claims.
- No opaque AI recommendation engine.
- No autonomous multi-agent loop in the scoring or risk pipeline.
